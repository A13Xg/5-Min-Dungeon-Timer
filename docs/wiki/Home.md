# 5-Minute Dungeon Companion Wiki

## What this app is for
This companion app supports real-time cooperative 5-Minute Dungeon sessions with a dedicated 5-minute stage timer, escalating urgency cues, and stage-by-stage card tracking.

## Stage guide
- Stage 1: 20 dungeon door cards
- Stage 2: 25 dungeon door cards
- Stage 3: 30 dungeon door cards
- Stage 4: 35 dungeon door cards
- Stage 5: 40 dungeon door cards

## Control summary
- **Start/Pause:** Main center button
- **Reset:** Left large button
- **Next:** Right large button
- **Back:** Small button under reset

## Settings
- Ambient music (default ON)
- Beeps (default ON)
- Keep awake (Wake Lock)
- Dark/Light mode
- Advanced/Simple graphics

## Browser compatibility
The app is fully static and runs entirely in-browser, making it suitable for GitHub Pages.
